link 与 @import 区别
  1. link 是 HTML 标签，@import 是css 方式
    • 兼容性: @import 低于 ie5 不支持
    • DOM操作
    • @import 模块化
    • @import 响应式
  2. 加载顺序
    • link 与页面同时加载，defered控制
    • @import 页面加载完后加载
  3. 优先级
    • link 优先级高于 @import
